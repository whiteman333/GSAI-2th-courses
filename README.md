# GSAI 大二课程资料整理

> 🎓 GSAI（中国人民大学高瓴人工智能学院）大二专业课学习资料整理
> 
> 📚 致力于帮助后来人更好地学习这些课程

## 📋 项目简介

本项目整理了GSAI大二两个学期的核心课程学习资料，包括课程实验、作业、课件、参考书籍等。希望能为后续学习这些课程的同学提供参考和帮助。

## 🗂️ 项目结构

```
GSAI-2th-courses/
├── First_sem/                    # 大二上学期
│   ├── AI Intro/                 # 人工智能引论
│   ├── Data structure and algorithm/  # 数据结构与算法
│   ├── ICS/                      # 计算机系统导论
│   ├── Math Basis/               # 人工智能数学基础
│   └── Probability and Statistics/ # 概率论与数理统计
└── Second_sem/                   # 大二下学期
    ├── Algorithm/                # 算法分析与设计
    ├── ML/                       # 机器学习基础
    └── Optimization/             # 最优化理论与方法
```

## 📚 课程详情

### 🔹 第一学期课程

#### 1. 人工智能引论 (AI Intro)
- **课程性质**: 专业必修课
- **主要内容**: AI基础概念、搜索算法、知识表示、机器学习基础
- **资料包含**: 课件、实验报告、参考资料

#### 2. 数据结构与算法 (Data Structure and Algorithm)
- **课程性质**: 专业核心课
- **主要内容**: 线性表、树、图、排序、查找算法
- **资料包含**: 
  - 4个完整的实验项目
  - 实验代码和报告
  - 算法实现和分析

#### 3. 计算机系统导论 (ICS)
- **课程性质**: 专业必修课
- **主要内容**: 计算机系统原理、汇编语言、操作系统基础
- **资料包含**: 课件、实验指导、参考书籍

#### 4. 人工智能数学基础 (Math Basis)
- **课程性质**: 专业基础课
- **主要内容**: 线性代数、概率论、微积分在AI中的应用
- **资料包含**: 教材、课件、习题解答

#### 5. 概率论与数理统计 (Probability and Statistics)
- **课程性质**: 数学基础课
- **主要内容**: 概率理论、统计推断、随机过程
- **资料包含**: 教材、习题、考试资料

### 🔹 第二学期课程

#### 1. 算法分析与设计 (Algorithm)
- **课程性质**: 专业核心课
- **主要内容**: 高级算法设计技巧、复杂度分析、NP问题
- **资料包含**: 算法实现、复杂度分析、经典问题解法

#### 2. 机器学习基础 (ML)
- **课程性质**: 专业核心课
- **主要内容**: 监督学习、无监督学习、深度学习基础
- **资料包含**: 理论课件、编程实验、项目代码

#### 3. 最优化理论与方法 (Optimization)
- **课程性质**: 专业选修课
- **主要内容**: 线性规划、非线性优化、凸优化
- **资料包含**: 数学推导、算法实现、应用案例

## 🚀 使用说明

### 环境要求
- 编程语言: C++, Python
- 开发环境: Visual Studio Code, CLion, PyCharm等
- 系统要求: Windows/Linux/macOS

### 获取资料
```bash
git clone [仓库地址]
cd GSAI-2th-courses
```

### 目录导航
- 每个课程文件夹都包含该课程的完整学习资料
- 实验代码通常在对应的`Labs/`或`code/`目录下
- 课件和参考资料在各课程的根目录中

## 📖 学习建议

### 第一学期重点
1. **数据结构与算法**: 这是后续所有课程的基础，务必打好基础
2. **计算机系统导论**: 理解计算机底层原理，对系统编程很重要
3. **数学基础**: 为机器学习和优化课程做准备

### 第二学期重点
1. **机器学习**: 结合理论与实践，多做项目
2. **算法设计**: 提高编程竞赛和面试能力
3. **最优化**: 深度学习的数学基础
